# Maths Heads · Hero image shopping list

## Easy route (recommended)

1. **Free Pexels key** → [pexels.com/api](https://www.pexels.com/api/) (log in, copy key — takes ~1 min).
2. In terminal:
   ```bash
   cd ~/Desktop/mathsheads
   cp .env.example .env    # paste key into .env
   node scripts/fetch-heroes.mjs
   ```
3. Refresh the app — title slides show heroes automatically.

The script downloads all **40** title heroes into `images/` and **34** hook images into `images/hooks/` (one per teach lesson, matched to the problem on slide 3).

**Optional:** add the same key to `.cursor/mcp.json` (replace `PASTE_YOUR_PEXELS_KEY_HERE`) so you can ask Cursor to search/replace individual images in chat.

---

**40 images** for Term 3 (one per lesson). Drop finished files into the `images/` folder using the exact filenames below — the title slide picks them up automatically.

## Specs (all images)

| Setting | Recommendation |
|--------|----------------|
| **Size** | At least **1600 × 1000 px** (landscape). Bigger is fine. |
| **Format** | **`.jpg`** from fetch script, or **`.webp`** if you source manually — update `lessons.js` if you change extension. |
| **Crop** | Landscape, subject centred. Faces/objects in the middle third (edges get cropped on narrow screens). |
| **Style** | Warm, bright, **kid-friendly**, real-life (not clip-art unless it’s a clear diagram). Diverse kids where people appear. |
| **Rights** | Your own photos, school-approved images, or royalty-free (Unsplash, Pexels, Pixabay, Canva stock). Keep a simple attribution log if required. |
| **Diagrams** | Lessons marked **diagram** use `object-fit: contain` — use clean graphics on a plain/dark background. |

## After you add images

1. Save files to `mathsheads/images/` with exact names above.
2. Refresh the browser — open a lesson → **slide 1 (Title)** shows the hero.
3. If an image is missing, you’ll see a placeholder until the file exists.

To change caption or swap a file, edit the `hero` block in `lessons.js`:

```javascript
hero: {
  src: 'images/w1-l1-pizza-night.webp',
  alt: 'Two pizzas cut into different numbers of slices',
  caption: 'Pizza night — who ate more?'
}
```

Add `diagram: true` for charts, nets, and graphics that should not be cropped.
